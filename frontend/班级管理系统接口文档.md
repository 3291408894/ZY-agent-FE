# 班级管理系统 — 接口说明文档

> Base URL: `http://localhost:8000/api/v1`  
> 认证方式: `Authorization: Bearer <access_token>`  
> 统一响应格式: `{"code": 0, "message": "ok", "data": {...}}`

---

## 一、教师端接口

### 1. 创建班级

```
POST /teacher/classes
```

> **鉴权**: `get_current_teacher` — 仅教师/管理员可访问

**请求体**:
```json
{
  "name": "高一(3)班数学",
  "grade": "高一",
  "subject": "数学",
  "description": "2026年秋季学期"
}
```

| 字段 | 类型 | 必填 | 长度 | 说明 |
|------|------|:--:|------|------|
| name | string | 是 | 1-100 | 班级名称 |
| grade | string | 是 | 1-50 | 年级 |
| subject | string | 是 | 1-50 | 学科 |
| description | string | 否 | 0-500 | 班级描述 |

**响应 (201)**:
```json
{
  "code": 0,
  "message": "班级创建成功",
  "data": {
    "id": "a1b2c3d4-...",
    "name": "高一(3)班数学",
    "grade": "高一",
    "subject": "数学",
    "description": "2026年秋季学期",
    "invite_code": "A3B7K9M2",
    "student_count": 0,
    "status": "active",
    "created_at": "2026-07-07T12:00:00Z",
    "updated_at": "2026-07-07T12:00:00Z"
  }
}
```

| 字段 | 说明 |
|------|------|
| invite_code | 8 位邀请码（排除 0/O/1/I/L），系统自动生成，全局唯一 |

---

### 2. 我的班级列表

```
GET /teacher/classes
```

> **鉴权**: `get_current_teacher`

**请求参数**: 无

**响应**:
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "classes": [
      {
        "id": "a1b2c3d4-...",
        "name": "高一(3)班数学",
        "grade": "高一",
        "subject": "数学",
        "description": "2026年秋季学期",
        "invite_code": "A3B7K9M2",
        "student_count": 25,
        "status": "active",
        "created_at": "2026-07-07T12:00:00Z",
        "updated_at": "2026-07-07T15:30:00Z"
      }
    ]
  }
}
```

> 列表按 `updated_at` 倒序；没有班级时返回空数组 `[]`；邀请码始终返回（教师可随时查看分享）。

### 4. 花名册

```
GET /teacher/classes/{class_id}/roster
```

> **鉴权**: `get_current_teacher`

**响应**:
```json
{
  "code": 0,
  "data": {
    "roster": [
      {
        "id": "cs-uuid-1",
        "student_id": "stu-uuid-1",
        "student_name": "小明",
        "joined_at": "2026-07-07T13:00:00Z"
      },
      {
        "id": "cs-uuid-2",
        "student_id": "stu-uuid-2",
        "student_name": "小红",
        "joined_at": "2026-07-07T14:00:00Z"
      }
    ]
  }
}
```

| 字段 | 说明 |
|------|------|
| id | ClassStudent 关联表主键 |
| student_id | 学生 User ID |
| student_name | 学生昵称（nickname） |
| joined_at | 加入时间，正序排列 |

---

### 5. 移除学生

```
DELETE /teacher/classes/{class_id}/students/{student_id}
```

> **鉴权**: `get_current_teacher`

| 参数 | 类型 | 说明 |
|------|------|------|
| class_id | string (UUID) | 路径参数 |
| student_id | string (UUID) | 要移除的学生 User ID |

**响应**:
```json
{
  "code": 0,
  "message": "学生已移除",
  "data": null
}
```

---

### 6. 重新生成邀请码

```
POST /teacher/classes/{class_id}/regenerate-code
```

> **鉴权**: `get_current_teacher`  
> **注意**: 旧码立即失效；归档班级不可操作

**响应**:
```json
{
  "code": 0,
  "message": "邀请码已重新生成",
  "data": {
    "invite_code": "B8G4PQ6X"
  }
}
```

---

### 7. 归档班级

```
PATCH /teacher/classes/{class_id}/archive
```

> **鉴权**: `get_current_teacher`  
> **注意**: 归档后不可新增作业，但历史数据仍可查看

**响应**:
```json
{
  "code": 0,
  "message": "班级已归档",
  "data": {
    "status": "archived"
  }
}
```

## 二、学生端接口

### 1. 通过邀请码加入班级

```
POST /student/classes/join
```

> **鉴权**: `get_current_user`（学生/教师均可加入）

**请求体**:
```json
{
  "invite_code": "A3B7K9M2"
}
```

| 字段 | 类型 | 必填 | 长度 | 说明 |
|------|------|:--:|------|------|
| invite_code | string | 是 | 8 | 邀请码，大小写不敏感，自动转为大写 |

**成功响应 (201)**:
```json
{
  "code": 0,
  "message": "加入成功",
  "data": {
    "success": true,
    "class_id": "a1b2c3d4-...",
    "class_name": "高一(3)班数学"
  }
}
```

**错误响应 (400)**:

| message | 场景 |
|------|------|
| "邀请码无效，请检查后重试" | 邀请码不存在 |
| "该班级已归档，无法加入" | 班级已归档 |
| "你已在该班级中，无需重复加入" | 重复加入同一班级 |

## 三、错误码

| code | HTTP | 说明 |
|------|------|------|
| 40001 | 400 | 参数校验失败（含邀请码无效） |
| 40101 | 401 | Token 过期 |
| 40102 | 401 | Token 无效 |
| 40301 | 403 | 非教师角色访问教师接口 |
| 40402 | 404 | 班级不存在或无权访问 |

## 四、前端对接指南

### 教师端页面结构

```
┌─ 班级管理首页 ─────────────────────────────────┐
│  [+ 创建班级]                                   │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐       │
│  │ 高一(3)班  │ │ 高二(1)班  │ │ 八年级语文 │       │
│  │ 数学       │ │ 物理       │ │           │       │
│  │ 25人 active│ │ 30人 active│ │ 18人 active│       │
│  │ 邀请码:    │ │ 邀请码:    │ │ 邀请码:    │       │
│  │ A3B7K9M2  │ │ B8G4PQ6X  │ │ ...       │       │
│  │ [复制][详情]│ │ [复制][详情]│ │           │       │
│  └──────────┘ └──────────┘ └──────────┘       │
└────────────────────────────────────────────────┘

点击卡片 → 班级详情页
  ├─ 班级信息 + 邀请码面板（大字体、一键复制、重新生成按钮）
  ├─ 花名册表格（学生姓名 / 加入时间 / 移除按钮）
  └─ [归档班级] 按钮（二次确认）
```

### 学生端页面结构

```
┌─ 我的班级 ─────────────────────────────────────┐
│  [+ 加入班级]                                   │
│  ┌──────────────────────────┐                  │
│  │ 高一(3)班数学             │                  │
│  │ 2026-07-07 加入           │                  │
│  │ [退出]                    │                  │
│  ├──────────────────────────┤                  │
│  │ 高一(3)班语文             │                  │
│  │ ...                      │                  │
│  └──────────────────────────┘                  │
└────────────────────────────────────────────────┘

点击 [+ 加入班级] → 弹窗
  ├─ 输入框：请输入 8 位邀请码
  ├─ [确认加入]
  └─ 成功后显示班级名称确认
```

### 关键交互细节

1. **邀请码展示**: 使用大号等宽字体，加 `cursor: copy` 一键复制到剪贴板
2. **重新生成**: 点击时弹 `ElMessageBox.confirm("重新生成后旧邀请码将立即失效，确定继续？")`
3. **归档确认**: 点击时弹 `ElMessageBox.confirm("归档后该班级将无法新增作业，确定归档？")`
4. **移除学生**: 点击时弹 `ElMessageBox.confirm("确定将该学生移出班级？")`
5. **加入班级**: 邀请码输入框自动转为大写，限制 8 位
6. **空状态**: 无班级时显示引导页："暂无班级，点击创建第一个班级吧"
